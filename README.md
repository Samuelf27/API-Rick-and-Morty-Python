## 📌 Sobre o projeto

Projeto didático de consumo de **API REST**: usando a biblioteca `requests`, o script faz requisições à [Rick and Morty API](https://rickandmortyapi.com/) e exibe os dados de personagens, episódios e localizações de forma simples e organizada.

## ✨ O que demonstra

- 🌐 Requisições HTTP a uma API pública (`requests`)
- 🔎 Filtragem e tratamento dos dados retornados (JSON)
- 🖥️ Exibição organizada dos resultados

## 🛠️ Tecnologias

Python · Requests · Rick and Morty API

## 🚀 Como executar

```bash
git clone https://github.com/Samuelf27/API-Rick-and-Morty-Python.git
cd API-Rick-and-Morty-Python

pip install -r requirements.txt

python main.py
```
